# APUE
Basic code for APUE

## For Mac OS:

Put APUE.h and error.c into //Applications/Xcode.app/Contents/Developer/Toolchains/XcodeDefault.xctoolchain/usr/include/

add at the end of the apue.h
#include "error.c"
#include "tellwait.c"
#include "prexit.c"
